Combinators

These functions are designed to manipulate the way functions are composed and executed. They are similar to code flow functions, except they focus more on execution style, rather than choices.

Functions

times
 N [P]  ->  ...

Executes P, N times

linrec
 [P] [T] [R1] [R2]  ->  ...

Executes P. If that yields true, executes T. Else executes R1, recurses, executes R2.

binrec
 [P] [T] [R1] [R2]  ->  ...

Executes P. If that yields true, executes T. Else uses R1 to produce two intermediates, recurses twice, then executes R2 (usually to combine their results).
